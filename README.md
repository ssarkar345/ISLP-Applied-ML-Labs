# ISLP-Applied-ML-Labs
Python implementations of statistical learning methods — regression, classification, clustering, dimensionality reduction, SVMs, and neural networks following James et al. An Introduction to Statistical Learning with Applications in Python (ISLP).

## Overview
Hands-on lab notebooks covering core machine learning and statistical learning methods, implemented in Python using scikit-learn, statsmodels, pandas, and matplotlib.

## Topics Covered
Linear and Logistic Regression
Resampling Methods (Cross-Validation, Bootstrap)
Regularization (Ridge, LASSO)
Tree-Based Methods (Decision Trees, Random Forests, Boosting)
Support Vector Machines
Unsupervised Learning (PCA, K-Means, Hierarchical Clustering)
Neural Networks and Deep Learning

## Tools and Libraries
Python · scikit-learn · statsmodels · pandas · NumPy · matplotlib · ISLP
## Notes
Labs completed as part of graduate coursework in Applied Machine Learning (Purdue University MSDS).
